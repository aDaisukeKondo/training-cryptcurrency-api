# training-cryptocurrency-api

training for some Cryptocurrency APIs

## Install

```sh
npm install
```

This code is tested with node.js 8.6.0

## Preparation

Input your api-keys and secrets to config.js

```js
{
  bitbank: {
    apiKey: 'your-api-key',
    secret: 'your-secret'
  },
  zaif: {
    apikey: 'your-api-key',
    secret: 'your-secret'
  },
  line: {
    token: 'your-token'
  }
}
```

## Run

```sh
npm start
```
